# Portfolio Website

![Portfolio Screenshot](/img/screenshot.png)

A modern responsive portfolio website built with:
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Font Awesome icons

## Features
✅ Dark/Light mode toggle  
✅ Interactive portfolio filter  
✅ Testimonials section  
✅ Responsive design  
✅ Smooth animations
